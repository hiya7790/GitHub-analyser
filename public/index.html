Fetch, analyze, and store GitHub user profiles. Get engagement scores, language breakdowns, repo stats & historical tracking.

API Endpoints
POST /api/analyze/:username Analyze a profile
GET /api/profiles List all profiles
GET /api/profiles/:username Get single profile
PUT /api/profiles/:username/refresh Refresh data
DELETE /api/profiles/:username Remove profile
GET /api/profiles/:username/repos User repos
GET /api/profiles/:username/history History
GET /api/profiles/compare?users=a,b,c Compare
GET /api/stats Stats dashboard
GET /api/github/rate-limit Rate limit
